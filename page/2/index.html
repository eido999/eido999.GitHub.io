计算机图形学 transformation

transformation（变换） 几何坐标系变换 用于顶点变换过程

BASIC transformation 将局部坐标系统一到整体 旋转 平移 投影 全部用矩阵的方式进行表达

  • 平面上的坐标变换（顶点）
    • 平移（scale）根据摸个参考位置为基点 按照具体比例进行放缩 基于比例矩阵相乘
    • 旋转 rotate 全部点用一个矩阵进行处理
  • 齐次坐标系（平面）
    • 平移变换 （坐标进行相应的加减）加法计算器（复杂）
    • 放缩 与比例相乘 乘法计算器
      • 为了让这两个计算能够统一为一种运算
      • 引入的w被称为齐次坐标
  • 组合变换 存在不满足交换律的组合变化
  • 3D变换
    • 四个维度的齐次坐标（三个点一个齐次）
    • 旋转使用三个矩阵进行性传传处理处理万向节死锁

View and perspective（视点与投影）

  • 视点坐标系转换

    • 以视点为原点 朝向眼睛为Z中正方向 水平向右为X周 竖直向上为Y轴正方向

  • 投影到屏幕面上

    • 具体投影方式为点投影

计算机网络 物理层

物理层：在传输媒体上如何进行数据流的传输，不仅指具体的传输媒体

WIFI可以穿透固体进行传播，但是在水中不能进行传播 潜艇只能用声纳进行交流。

数据通信系统模型

  • 组件

    • 源系统 发送端 （网卡即调制解调器）
    • 传输网络（公用电话网）
    • 目的系统

  • 术语

    • 数据（data）消息实体
    • 信号（signal）数据的电磁或电气表现
    • 模拟信号（Analogous S）
    • 数字信号（Digital S）

  • 信道

    • 晁某一个方向传送信息的媒体
    • 单项通信 （电视 广播）
    • 双向交替通信 不能同时发送 不能同时接受（对讲机）
    • 双向同时通信 可以同时发送 同时接受

  • 常用编码方式

归零制：在一个时间片内存在几个向下脉冲就有几个零

物理层：编码 调制（OFDM不完全是调制） 整层

  • 调制

    调制信号使之符合使用要求

    • 信道的极限容量
      • 信道在传输信号过程中一定会存在失真
      • 传输速率越高 距离越远 媒体质量越差 失真越严重

  • 香农公式

  • C = Width * log2（1+ Signal/Noise）

  • 信噪比越大 同一信道极限传输功率越高

  • 只要未达到极限传输速率就有无差错传输方式

微波炉与WIFI均使用2.4G微波

ADSL：可以根据用户的实时需求保证用户拥有较高的传输速率

HFC网 光纤同轴电缆复合网络 其中管线负责主干网络传输。

电缆调制解调器：HFC网中用户端对数据进行处理的复杂调制解调器。

FTTx技术 FIber to the 整栋楼的用户共享同一根光纤 好处是高速传输。

虽然每户摊到得得流量生成较高，但是在集体大量使用时候实际流量会小于声称值

多种通信协议的产生 ： WIFI 蓝牙 4G

计算机图形学-图形流水线

图形流水线

3D geometry ：致顶点构成的3D几何体的相对位置关系描述

3D material ： 表示对应的几何面的材质

lights： 光的颜色 能量 光源的形状。

camera ： 摄像机的设定

一个简单的图形流水线步骤示意图

应用请求提出在CPU 而之后的摄像机运算与光栅化都在GPU上

首先CPU按照需求将环境中图形篇有层次进行分割

然后转到GPU GPU首先把元素按照三维空间的正确位置进行摆放之后进行按材质打光

特点：

  • I/O规模较大

application：

将需求解构为构成图形的基本3D元素 点 线 环 面

  • 根据线的环向用右手定则确定正面

openGL （CPU与GPU通信接口）

  • 提供了10种初始形状。

顶点：

  • 包含最基本的位置信息（xyz）

  • 包含颜色（RGB设定）有的还添加了透明度

  • 顶点的法向量 （表示点附近所形成表面的法向量方向 主要用于曲面）在进行光照计算时候同样注意

  • 纹理（texture）：某一表面图案与每个顶点的二维平面关系

光栅化：

  • 图像是由像素构成的 而在三维空间上的连续表达要转化成图像需要进行光栅化的扫描转换。
  • 在这个过程中要注意像素点的颜色变化
  • pixel（像素） fragment（并非在屏幕上而是原始数据转换时产生的原始像素分割 实际上要更为丰富）

3D图像的流水线：

除了第一阶段外均在GPU中进行。

传统GPU架构：

  • vertex processer（顶点处理器） 固定定点在坐A标系中位置
  • rasterizer 光栅化
  • fragment processer 透明度 着色等
  • 输出

coordinate system（用右手定则进行建系）

  • 世界坐标系
  • 观察坐标系
  • object 坐标系

model transform

将每个物体组件的相对位置关系转换为整体使用的世界坐标系

投影画面是很大的，但是窗口画面只有固定部分

之后会考虑到观察角度进一步转化为观察者坐标系。

可见性（出现于光栅化部分）：

  • 早期：画家算法

    • 由远及近 有后往前进行渲染
    • 在对远处物体进行渲染后立即对近处物体进行重新渲染
    • 对于互相拥有前后关系的物体无法实现正常渲染

  • Z-buffer

    • 将整体画面进行分块 观察局部画面是否满足画家算法

    • 但是无法完全解决问题

    • 将问题下放至每一个像素的问题（目前基本使用方法）

    • 对每个像素的远近关系进行实现

    • 根据观察者视点来确定每一个像素点的Z值

    • 同一观察线上选取Z值最小值像素点进行更新（光栅化环节进行完成）

    • 尽管涉及到每一个像素点，但是全部是浮点运算（不存在不可并行部分）

酷女人第一弹！

东方刚欲异闻在今年上半年宣布了将于暑期退出NS版并添加了道具路线与饕餮尤魔个人路线的消息。对于为刚欲异闻本体略显单薄的路线数感到可惜的玩家们来说，NS版惯例的路线追加提供了一次体验更全面剧情的机会。

而本作最终BOSS饕餮尤魔，在经历鬼形兽摸鱼，刚欲异闻延迟推出等一串蹲坑事件后终于迎来了格斗作惯例的个人路线。

承接重量级整数作东方鬼形兽，由前一部作品的黑幕摩多罗隐岐奈一手引导，揭晓鬼形兽中刚欲同盟总长，饕餮的动物灵饕餮尤魔神秘消失的原因，并为更具现实气息的东方虹龙洞做铺垫，本作相比其他格斗作在剧情上与主线的联系更加紧密，可以猜测在鬼形兽甚至是天空璋时期就已经做好了剧情规划。出色的剧情安排与美术风格让我迷上了这一部东方作品，给了我深挖剧情的动力。

石油异变是一场牵扯到人妖鬼神，波及幻想乡全境的重量级异变，黄昏边境在宣传这部作品时使用了以前从没用过的形容，“灾难性”。剧情中也出现了知道血池地狱真相的命莲寺为了及时封印血池地狱在不知会幻想乡其余势力的情况下凿穿三途川的情节，从一个侧面反映了本次异变的严重性。

这次异变的黑幕，正是在明白石油的危险性，了解饕餮尤魔无法被常规手段击败，甚至已经想好饕餮尤魔对策的情况下满怀热情地投入到将真身是旧血池地狱的血液的石油引向地表的事业中，成功地整出了一般神整不出的节目效果。

轮椅战神，新世代锅王，究极的绝对乐子秘神，强势登场后存在感极强的摩多罗隐岐奈是这场异变毋庸置疑是这场异变的绝对主角。

相比谜语人八云紫跟正经人茨木华扇，唯我独尊的摩多罗很明显更为强势，个人风格更强，我在写文稿的过程中也几次三番的沉浸在对这位秘神的狂热崇拜中，回过神来的时候已经离题万里，纸上都是赞美之词，就像是身处在狂欲之海中的饕餮尤魔，喝高了让人狂热的石油，如秘神背后的童子一般失去了理智。抛开行为深处的逻辑不提，这也是个让人恨不起来的角色。

但是刚欲异闻真的只是摩多罗一个人的操线起舞吗？这样的想法未免有些过于傲慢，站在狂热信徒的角度来看或许如此。过去有幻想乡不过是紫的金鱼缸的说法，如今有摩多罗的万年黑幕，幻想乡的的平衡似乎永远都是建立在某种绝对力量（BBA）之后。从最近的作品比如在人里开设妖怪酒馆、座敷童子实现远程办公的醉蝶华，妖怪大佬纷纷白给的智灵奇传，以及茨歌仙最经典的守矢，河童与天狗三方共建缆车桥段都可以看出，幻想乡之间的各个势力正在有机的结合在一起，而不再是梦月抄中简单的必要关系与宗教战争那种直白的对立关系。有别于过去的互相遏制，一种全新的平衡正在到来。刚欲异闻中的八坂神奈子就是一个例子，线路结尾的洩矢诹访子也是威严满满，本作的守矢势可以说是神德充沛。

故事回到一切的最开始，自机们在畜生界被各路大佬暴打的时候，临阵摸鱼的饕餮尤魔离开了畜生界，并通过某种手段到达了位于旧地狱深处的旧血池地狱。现有的剧情里并没有直接讲明因为过于可怕被地狱管理者们封印的旧血池地狱是如何解开了封印。摩多罗发现了旧血池地狱封印被破除，贪求着石油力量的饕餮盘踞在地底，并因为自身的能力逐渐受到石油中的诅咒影响。为了处理这一问题将石油引导了地上，揭开了石油异变的序章。

在全部势力中最早行动的是与地底有着深刻渊源的命莲寺，过去曾被封印在地底的圣辇船妖怪们从一开始就知道石油的真面目，认为封印旧血池地狱刻不容缓，到达旧地狱最深处需要穿过高温的旧灼热地狱，在圣的授意下村纱水蜜凿穿了三途川，试图用无量之水冷却旧灼热地狱，这也是标题中被水淹没的沉愁地狱的由来。这一行为同时影响了守矢的核融合炉与地狱方面，由于事态紧急并未事先通知。

三途川泛滥的河水开始向地底涌去的同时，由于石油涌出造成的核融合炉故障惊动了守矢神社的山神八坂神奈子，风神亲自赶往地底掌控事态，而靠着石油大发了一笔的依神姐妹这个时候或许正在旧地狱温泉大闹。各条线路并非平行世界而是同一个事件的不同侧面，刚欲异闻这种与以往格斗作不同的叙事方式让复盘这场异变的过程变得很有趣。

人类方面站出来的自然还是红白黑白两只自机，两条路线有一定程度的并行关系。刚刚到达魔法森林的她们对于到底发生了什么可以说是完全不了解，只是凭着正常的思路向着地底进行探查，两人打了打意思意思，算是为进入地底热身，魔理沙在此之后恰好碰上了正在赶往地底的船长，两人进行了友好的谜语人交流之后先后穿过地灵虹洞。在这里的时间顺序大概是灵梦、魔理沙、神奈子、船长，这一点可以用神奈子投喂因魔理沙不能吃而饿肚子的山女与小伞见到船长证实自己跟灵梦所说的猜测来证明。

呆在地灵虹洞的小伞跟山女也是在本次异变中十分有趣的角色，小妖怪小伞想要得到一点石油，而地底的土蜘蛛妖怪山女却知道石油的真面目并感到危险，并在石油再次喷发后颇具深意地说出盖上盖子对旧血池地狱是不起作用的。小妖怪因为石油的力量而欢喜，知晓石油真面目的妖怪断言过去封印旧血池地狱的做法是治标不治本，这两种对于石油的态度也很有趣。

正式进入地底后的第一层是旧地狱温泉，关键人物是以总管身份华丽登场的星熊勇仪。知晓石油真相的鬼王在面对前来调查石油的人类，靠石油赚来的财富大肆挥霍的依神姐妹表现出了不同的态度。

面对略有些下头的暴发户依神姐妹，鬼王没有点出为她们带来财富的正是包含诅咒的石油，而是招待了她们，不过没有也接受这对姐妹包场的要求。而对于提出调查的人类，勇仪在不透漏石油的真面目的情况下说明了石油的危险性，以鬼的风格大闹了一场之后放行了自机，告诫人类不可将旧地狱的秘密外传。知晓石油的危险，却不反感使用石油赚来的金钱的依神姐妹。一面说旧血池地狱对于旧地狱是绝密，另一面却又不断透露有关石油的信息，败北后爽快的指明前往石油之海的道路。石油在温泉喷发之后也只是吃了一惊，与芙兰交手之后就又恢复了常态，来者不拒，去者不留，说着混沌即安泰的鬼王似乎只是希望与来者交手，究竟是单纯的粗神经还是有着更深的考量。在饕餮路线中我们应该能够看到更有趣的描述。

在船长还在与赖在温泉街不走的依神姐妹拉扯的时候，灵梦与神奈子在核融合炉爽快地打了一架。由石油穿过旧地狱到达地面察觉到事件可能有内情的风神试图将灵梦赶回地面，可惜灵梦并不买账。解开误会后，灵梦继续向地底深入并穿过了灼热地狱，而神奈子去确认旧地狱是否有石油喷发。

也就是在这个时间点，无穷无尽的三途川之水淹没了灼热地狱。

从旧灼热地狱复燃的时间点来看，只有灵梦穿过的是还不曾熄灭的灼热地狱，其他角色到达石油之海的时间可能都在灵梦之后，在与饕餮交战的时间遥遥领先其他可用角色，很有可能在其他人还没下到石油之海的时候就已经被饕餮送回地上。剩下的人当中神奈子与魔理沙是在旧灼热地狱熄灭的时间段通过旧灼热地狱，依神姐妹与船长都是在旧灼热地狱复燃之后才通过，依神姐妹后来又碰上了刚刚与饕餮达成协议的神奈子，还被摩多罗用门带到了红魔馆。按照对战顺序，可以猜测时间上到达石油之海的顺序为灵梦、魔理沙、神奈子、依神姐妹、船长，与饕餮的对战顺序则是灵梦、魔理沙、神奈子、船长、依神姐妹。

把守着旧灼热地狱的是继承了八咫乌之力的地狱鸦，灵乌路空，而在旧灼热地狱熄灭期间登场的则是自三途川而来的二羽渡神庭渡久侘歌，两位鸟脑袋都是很清爽的角色，只是在清理融合炉中异物的阿空虽然知道地底有很危险的生物，但是并没有什么兴趣，认真地进行着污染物的清除。久侘歌作为地狱口岸的管理者前来调查三途川所发生的事故，却并没有碰上凿穿三途川的真犯人船长，与神奈子短暂的交手后听闻她要前往地底调查，努力阻止未果。这里我们也能看出来，地狱的管理者们知道旧血池地狱的存在并把进入旧血池地狱认定为危险举动，但是新来到地底的年轻妖怪有可能不知道。或许是着急离开即将复燃的旧灼热地狱，俗气的依神姐妹并没有引起二羽渡神过多注意。在芙兰朵露的路线中我们看到二羽渡神站在本应复燃的旧灼热地狱中，望着三途河水所流向的地狱深处，感叹连废墟中残存的仇怨都要抹除的无情。为什么她要为这颗被抹掉的定时炸弹感叹呢？这一点也引人深思。

灵梦路线对坠落进入石油之海的过程做了比较详细的描述。

在黑暗中不断下坠，双脚落在地面上时发现自己已经身处无边的石油海中，耳边回响着恶毒的歌谣：“一掘为了钱财，二掘为了国家，三掘为了憎恨”，寻声望去，迎上来的是一双血红的眼睛，还有诡异的笑容。浸在黑色的原油里，面前的白发妖怪透漏着狂气，让人有一种窒息的感觉。

这一作的人物服饰都很有特色，给人印象最深的应该是勇仪跟船长，依神姐妹的穿搭则参考了说到石油与金钱就绝对绕不开的中东石油王，配合中东风的改编曲更是令人两眼一黑。原创角色饕餮尤魔本体是尖耳白毛红瞳萝莉，羊角与服饰很明显地参考了兽面纹。另外耳环，尖牙，方形的眼瞳跟与娇小的身体呈现出反差的大勺子也是萌点，最重要的一点是什么呢，她是个正经露肩角色，一个不穿袖筒的露肩角色，头上还有蝴蝶结，真的太棒了，一伯分。

对异变的真相了解不足，也无法处理石油的人类们无法成为这次异变的解决者。

对于没见识到石油之海真面目的人类来说，做客此地的饕餮尤魔就足以造成很深的心理创伤，白给的主角组被挥舞着勺子的神秘妖怪借着上升的气流送回地面的间歇泉。鬼形兽中就见识到畜生可怕的自机二人，灵梦对没能完全解决的异变真相一头雾水，被饕餮尤魔边狂笑边采掘石油的狂气景象震撼的沙沙则产生了畜生PTSD。芙兰C路线中出现在石油之海的灵梦再次返回地底并与饕餮交战，这一点给了我些许的希望，实装的道具路线大概率是steam版中无法选择的刚欲挑战模式，不过也存在补上这条灵梦重回地底的线路的可能，毕竟在道具选择画面中出现了一直隐藏在幕后的摩多罗。失落，犹豫的灵梦在与芙兰的对战中找回了自我，猜出了吸收了自己攻击的饕餮并不是让石油喷出的犯人，恍惚中察觉到这场异变似乎另有隐情，可惜人类们的故事就此落幕了。

不同于人类，拥有强大力量，在与饕餮的战斗中知晓了血池地狱真相的妖怪与神明们又是怎样的呢？

山神八坂神奈子，想要管理石油。在灵梦面前说自己没去过旧血池地狱，实际上却是在向灵梦隐藏石油之海的真相，探求着石油的畜生让她想起了外界，正如诹访子所说，燃烧本是生物的石油受到诅咒是无法避免的，而面对苏醒的旧血池地狱只有委托不断探求着石油的饕餮才能处理掉，因此她选择了与饕餮尤魔签订秘密契约来管理石油。
